Geolocation Security

IPStack API

Real-time IP geolocation and threat level assessment

IPStack provides a powerful, real-time IP to geolocation API capable of looking up accurate location data and assessing security threats from risky IP addresses. Perfect for personalizing user experiences and implementing security measures.

Key Features

  • Lookup country, region, city, timezone, currency, and connection info
  • Bulk lookup (up to 50 IPs) via comma-separated calls
  • Hostname resolution & security checks (proxy/Tor/threat detection)
  • Return JSON or XML with JSONP callbacks
  • Multi-language support
Use Case:

Personalize user experiences by displaying location-specific content, implement geo-blocking for security, or provide localized language options based on visitor location.

cURL curl 'https://api.ipstack.com/134.201.250.155?access_key=YOUR_ACCESS_KEY'

Finance Data

Fixer API

Real-time & historical foreign exchange rates

A simple and lightweight API for real-time exchange rate data for 170 world currencies, updated every 60 seconds. Includes historical data available back to January 1, 1999.

Key Features

  • Latest exchange rates and historical rates (EOD)
  • Currency conversion endpoint
  • Time-series data (up to 365 days) & fluctuation data
  • Supports changing base currency and selecting specific symbols
  • Bank-level security (256-bit SSL), CORS, JSONP & ETag cache support
Use Case:

Build financial dashboards or analytics tools needing up-to-the-minute exchange rates with historical comparisons. Perfect for e-commerce, fintech apps, and investment platforms.

cURL curl "https://data.fixer.io/api/latest?access_key=YOUR_API_KEY&symbols=USD,CAD,JPY"

Validation Security

Numverify API

Validate phone numbers for 232 countries worldwide

A full-featured RESTful JSON API for national and international phone number validation and information lookup. Numbers are processed in real-time and cross-checked with the latest international numbering plan databases.

Key Features

  • Validate number, country code, format, and carrier
  • Supports E.164, international and national formats
  • Detect line type (mobile/landline)
  • Include location information
  • Coverage for 232 countries
Use Case:

Verify phone numbers during user registration, protect against fraud by validating carrier information, and optimize SMS delivery by identifying line types.

cURL curl --request GET 'https://api.apilayer.com/number_verification/validate?number=14158586273' \ --header 'apikey: YOUR_API_KEY'

Finance Data

Marketstack API

Real-time and historical stock market data

RESTful JSON API providing real-time, intraday and historical stock market data from 30,000+ tickers across 70+ global exchanges, with over 30 years of historical data.

Key Features

  • Real-time stock quotes and intraday prices (1 min, 5 min intervals)
  • End-of-day (EOD) historical data for 30+ years
  • Coverage of 500K+ tickers including global exchanges, indices, currencies, bonds, ETFs
  • Supports splits, dividends, company data, timezones
  • Secure (256-bit HTTPS) and easy authentication
Use Case:

Power trading dashboards, portfolio trackers or investment tools with live charts and historic trends. Ideal for fintech applications and financial analytics.

cURL curl "https://api.marketstack.com/v2/eod?access_key=YOUR_ACCESS_KEY&symbols=AAPL"

Aviationstack API

Real-time and historical global flight tracking

RESTful JSON API delivering real-time flight status, historical data, route, airline, airport, and aircraft information. Covers 13,000+ airlines and 10,000+ airports in 250+ countries.

Security Content Moderation

Violence Detection API

Flag violent images automatically

Automate content approval processes by detecting violent imagery. Computes a score on a 1 to 5 scale, where 5 means the content most likely contains violence while a score close to 1 implies the content is safe to publish.

Key Features

  • Automated violence detection in images
  • Score-based system (1-5 scale)
  • Context-aware analysis
  • Real-time processing
  • URL or file upload support
Use Case:

Moderate user-uploaded images on social platforms, content sharing sites, or AI-generated image platforms to ensure community safety and platform compliance.

cURL curl --request GET \ --url 'https://api.apilayer.com/violence_detection/url?url={url}' \ --header 'apikey: YOUR_API_KEY'

Content Moderation Security

Bad Words API

Advanced profanity filter for content moderation

Intelligent profanity detection system using English phonetics to identify and filter inappropriate language. Goes beyond simple word matching by analyzing how words sound, making it highly effective at catching variations.

Key Features

  • Phonetic-based detection (catches variations like "fck", "frck")
  • Identify swear words and their locations in text
  • Replace profanity with custom censor characters
  • Real-time processing
  • Detailed reports with word positions and deviations
Use Case:

Moderate user-generated content in forums and comment sections, filter chat messages in real-time apps, and ensure AI assistant responses are safe for all users.

cURL curl --request POST \ --url 'https://api.apilayer.com/bad_words?censor_character=*' \ --header 'apikey: YOUR_API_KEY' \ --data-raw 'Your text content here'

Media Utility

Social Media Assets Generator API

Resize and optimize images for social media platforms

Automatically resizes and creates appropriately formatted images for various social networks including Instagram, Facebook, Twitter, LinkedIn, YouTube, Pinterest and Tumblr.

Key Features

  • Support for all major social platforms
  • Multiple orientations: square, landscape, portrait, profile, story
  • Automatic center-point cropping
  • File upload or URL processing
  • Returns optimized image URLs
Use Case:

Automatically generate properly sized social media graphics for marketing campaigns, create consistent brand images across platforms, and save time on manual image resizing.

News Search

World News API

Global news with semantic search

Access thousands of news sources in over 50 languages from more than 150 countries worldwide. Features semantic search capability that understands context and variations in terminology.

Key Features

  • Semantic search (understands "USA", "US", "United States")
